# RikTech Coder

Mobile-like React frontend app (no backend). Ready to deploy to Vercel as a static site.

## Quick start
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run dev server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```
4. Deploy to Vercel:
   - Push the project to a Git repo (GitHub/GitLab).
   - In Vercel, import project and use commands:
     - Build command: `npm run build`
     - Output directory: `dist`

## Notes
- The app is fully client-side (no backend).
- Uses Font Awesome for bottom navigation icons.
